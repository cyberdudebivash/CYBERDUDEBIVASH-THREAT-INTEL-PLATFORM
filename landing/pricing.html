FAQ

Common Questions

How do I get my API key after payment?
Your API key is provisioned automatically within seconds of successful payment. It is delivered to your email via SendGrid. You can also check the /api/v1/onboard endpoint for quick-start documentation.
What payment methods are accepted?
International: Stripe accepts all major credit/debit cards and many local payment methods. India/APAC: Razorpay accepts UPI, Netbanking, credit/debit cards, and wallets. Gumroad is also available for one-time purchases.
Can I test the API before subscribing?
Yes. The Free tier requires no signup or API key. Call GET /api/v1/intel/latest directly. Use demo key demo-pro-key-1111 to test Pro tier features without payment.
How is the data updated?
The pipeline runs every 6 hours via GitHub Actions. Each run: fetches from 10+ threat intel sources, enriches via 12 APEX AI engines, exports STIX 2.1 bundles, and publishes to the manifest and dashboard. No manual intervention.
What formats does the API return?
All endpoints return JSON. The /intel/feed endpoint includes full APEX enrichment, IOC counts, MITRE tactics, CVSS/EPSS scores, and campaign IDs. STIX 2.1 bundles with x-cdb-apex-1 custom extension are available for Pro+ tiers.
Can I cancel anytime?
Yes. Cancel via Stripe or Razorpay dashboard anytime. Your key remains active until end of billing period. On cancellation, the key is automatically revoked and you revert to Free tier.
