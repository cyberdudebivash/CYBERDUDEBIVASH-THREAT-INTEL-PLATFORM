PYTHON — JWT Auth Flow 
import requests

BASE = "https://intel.cyberdudebivash.com"

# Step 1: Get JWT
resp = requests.post(f"{BASE}/api/auth/token",
    json={"api_key": "YOUR_API_KEY_HERE"})
token = resp.json()["access_token"]

# Step 2: Use JWT Bearer
headers = {"Authorization": f"Bearer {token}"}
feed = requests.get(f"{BASE}/api/v1/intel/feed",
    headers=headers, params={"page": 1, "limit": 50})
print(feed.json())
JAVASCRIPT — Fetch Client (Browser / Node 18+)

  fetch(`${BASE}${path}`, auth ? { headers: {'X-API-Key': KEY} } : {})
    .then(r => r.json());
