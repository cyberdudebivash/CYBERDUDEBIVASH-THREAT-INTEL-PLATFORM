SENTINEL APEX API Reference v185.0

The SENTINEL APEX API provides programmatic access to real-time threat intelligence, STIX 2.1 bundles, analyst dossiers, IOC lookups, and SIEM dispatch integrations. Base URL:

https://intel.cyberdudebivash.com/api/

📘 All API responses are JSON. STIX endpoints return STIX 2.1-compliant bundles. Authentication via X-API-Key header is required for all endpoints except /api/health. Free tier provides 100 requests/day to public endpoints.

package main

import (
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

const (
    apiKey = "YOUR_API_KEY"
    base   = "https://intel.cyberdudebivash.com/api"
)

func main() {
    req, _ := http.NewRequest("GET", base+"/feed?limit=10&severity=CRITICAL", nil)
    req.Header.Set("X-API-Key", apiKey)
    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    body, _ := io.ReadAll(resp.Body)
    var advisories []map[string]interface{}
    json.Unmarshal(body, &advisories)
    for _, adv := range advisories {
        fmt.Printf("[%s] %s — CVSS:%v\n", adv["severity"], adv["title"], adv["cvss_score"])
    }
}

Authentication

All authenticated endpoints require an API key passed in the X-API-Key header. Request your API key at pricing.html.
