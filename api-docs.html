Quick Start

Base URL for all API endpoints:

https://intel.cyberdudebivash.com

1. Get an API key

Register at /get-api-key.html. Free tier keys are issued instantly with no credit card required.

2. Make your first request

# Public preview — no auth required
curl https://intel.cyberdudebivash.com/api/preview

# Authenticated feed request
curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \
     https://intel.cyberdudebivash.com/api/feed

# API key auth (legacy)
curl -H "X-API-Key: CDB-your-api-key-here" \
     https://intel.cyberdudebivash.com/api/feed

3. Parse the response

import requests

token = "YOUR_JWT_TOKEN"
headers = {"Authorization": f"Bearer {token}"}

resp = requests.get(
    "https://intel.cyberdudebivash.com/api/feed",
    headers=headers
)
data = resp.json()
print(f"Total advisories: {data['total']}")
for item in data['items'][:5]:
    print(f"{item['severity']:8s} | {item['title']}")

Authentication

SENTINEL APEX uses JWT (HS256) for authentication. Obtain a token via /auth/login, then pass it as a Bearer token in the Authorization header.

Free tier endpoints (/api/preview, /api/health) require no authentication.

SDKs & Integrations

Python SDK

# Install
pip install requests

# Usage
import requests

class SentinelApexClient:
    BASE = "https://intel.cyberdudebivash.com"

    def __init__(self, token):
        self.headers = {"Authorization": f"Bearer {token}"}

    def get_feed(self, severity=None, kev_only=False):
        params = {}
        if severity: params["severity"] = severity
        if kev_only: params["kev"] = "true"
        r = requests.get(f"{self.BASE}/api/feed", headers=self.headers, params=params)
        r.raise_for_status()
        return r.json()["items"]

client = SentinelApexClient("YOUR_JWT_TOKEN")
criticals = client.get_feed(severity="CRITICAL", kev_only=True)
print(f"Active KEV criticals: {len(criticals)}")
