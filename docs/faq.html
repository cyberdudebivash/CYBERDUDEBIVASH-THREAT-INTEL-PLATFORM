Payments & Billing
How do I pay? What payment methods do you accept? +
We accept: UPI (GPay, PhonePe, Paytm, BHIM, Amazon Pay), Bank NEFT/IMPS, PayPal, Crypto (USDT TRC20/ERC20/BEP20, Bitcoin), and Amazon Pay (via UPI). Visit the Payment Gateway to pay. Stripe is not currently available.
I paid. When will I get my API key? +
Within 2 hours of payment verification. After you submit your Transaction ID / UTR in the Payment Tracker, we verify the payment (usually 15–30 minutes) and email your API key. For urgent activation, WhatsApp +91 8179881447 with your payment screenshot.
I submitted my UTR but haven't received my API key. What do I do? +
First, check your spam/junk folder. If not there, WhatsApp +91 8179881447 with your Reference ID (SA-YYYYMMDD-XXXX) and payment details. We'll resolve it within 30 minutes during business hours (9am–7pm IST).
Can I get a GST invoice? +
Yes. A GST-compliant invoice is attached to every API key delivery email. GSTIN: 21ARKPN8270G1ZP. If you need a proforma invoice before payment (common for enterprise procurement), email bivash@cyberdudebivash.com with subject "PROFORMA INVOICE REQUEST".
How do I renew my subscription? +
We'll email you at D-14, D-7, and D-3 before expiry with a renewal link. To renew: pay the same amount via any payment method → submit your new Transaction ID at PAYMENT-GATEWAY.html → your key is extended within 2 hours. Your API key stays the same — no reconfiguration needed.
Is there a free trial? +
Yes. The Community plan (₹0/free) gives you 100 API calls/day with access to the top threat feed. A 7-day PRO trial is available on request — email bivash@cyberdudebivash.com with your company and use case. Trial keys are issued within 2 hours.
What happens if I don't renew on time? +
Your API key enters a 3-day grace period after expiry — it continues to work during this window. After the grace period, the key is suspended (returns 401). Your data and subscription history are preserved for 30 days. To reactivate, simply renew and submit payment.
API & Technical
How do I authenticate API requests? +
Use Bearer token authentication in the Authorization header: Authorization: Bearer YOUR_API_KEY. Example: curl -H "Authorization: Bearer SA-PRO-ABC123" https://intel.cyberdudebivash.com/api/intel
What are the API rate limits? +
Community: 100 calls/day
PRO: 5,000 calls/day
Enterprise: 50,000 calls/day
MSSP: 500,000 calls/day

Check your remaining quota at: GET /api/quota. When you exceed the limit, the API returns HTTP 429 with a Retry-After header. Quotas reset daily at midnight UTC.
My API key returns 401 Unauthorized. What's wrong? +
Common causes: (1) Key copied incorrectly — check for extra spaces or line breaks. (2) Key not yet activated — if you submitted payment, allow up to 2 hours. (3) Subscription expired — renew at PAYMENT-GATEWAY.html. (4) Key was suspended — contact bivash@cyberdudebivash.com.
How do I integrate with Microsoft Sentinel / Splunk / Elastic / QRadar? +
Full platform-specific integration guides are at onboarding.html. We support STIX 2.1/TAXII, KQL rules, SPL rules, EQL rules, Sigma, and YARA. For enterprise onboarding assistance, email bivash@cyberdudebivash.com.
Can I rotate my API key? +
Yes. Email bivash@cyberdudebivash.com with subject "KEY ROTATION REQUEST" and your Reference ID. A new key is issued and your old key remains valid for 7 days to allow smooth transition.
