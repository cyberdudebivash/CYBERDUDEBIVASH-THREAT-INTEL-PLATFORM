Step 2
Make Your First API Call
Replace YOUR_KEY with your actual API key.

cURL (Terminal)

# Fetch latest critical severity threats curl -H "Authorization: Bearer YOUR_KEY" \ "https://intel.cyberdudebivash.com/api/intel?severity=critical&limit=5"

Python

import requests API_KEY = "YOUR_KEY" headers = {"Authorization": f"Bearer {API_KEY}"} response = requests.get( "https://intel.cyberdudebivash.com/api/intel", headers=headers, params={"severity": "critical", "limit": 10} ) data = response.json() print(f"Retrieved {data['count']} threats") for threat in data["intel"]: print(f" [{threat['severity']}] {threat['title']}")

JavaScript (Node.js / Browser)

const API_KEY = 'YOUR_KEY'; const response = await fetch( 'https://intel.cyberdudebivash.com/api/intel?severity=critical&limit=10', { headers: { 'Authorization': `Bearer ${API_KEY}` } } ); const data = await response.json(); console.log(`Retrieved ${data.count} threats`);
