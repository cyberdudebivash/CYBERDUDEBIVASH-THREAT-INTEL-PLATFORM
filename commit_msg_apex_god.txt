KNOWLEDGE BASE (KB)
  KB.MITIGATIONS — 7 threat types × 6 steps each
    ransomware, phishing, apt, exploit, malware, vulnerability, generic
  KB.DETECTIONS — SIGMA + YARA + Suricata per threat type
    ransomware, phishing, exploit, generic templates
  KB.IMPACT_MATRIX — 4 impact domains × 4 severity levels
    financial, operational, reputational, compliance

ANALYSIS ENGINES
  computeRiskMatrix(item) — 6-factor weighted composite score
    CVSS 28% + EPSS 20% + KEV 18% + IOC Density 14% + TTP 12% + Actor 8%
  computePredictions(item) — exploitation window, exploit probability,
    patch urgency SLA, spread probability timeline
  computeRecommendations(item) — IMMEDIATE/7-DAY/30-DAY action plans
  getThreatType(item) — classifies into 7 threat type buckets

HTML BUILDERS (6 tabs)
  buildRiskTab()   — 6-factor matrix + composite score + verdict badge
  buildMitiTab()   — numbered mitigation steps, priority badges
  buildPredTab()   — 4 prediction KPI cards + spread probability bar
  buildRecTab()    — tiered recommendations by urgency timeframe
  buildImpactTab() — 2×2 impact domain grid with level badges
  buildDetectTab() — SIGMA/YARA/Suricata rule viewer (Enterprise only)

TIER GATING
  Free:       Risk tab + 2 mitigations + IMMEDIATE recs only + paywalls
  Pro:        All mitigations + full 3-tier recommendations
  Enterprise: Full access including Detection Rules tab

RUNTIME ENGINE
  Fetch interceptor — wraps window.fetch to capture all feed items into
    window.__AGOD_MANIFEST keyed by stix_id (zero-latency lookup)
  processCard(el) — looks up item by data-stix-id, falls back to card
    attributes (severity, score, title) for cards without manifest data
  startObserver() — MutationObserver on #threat-grid processes all
    existing + future cards (handles live fetch card appends)
  window.agodToggle() / window.agodTab() — global event handlers
  window.AGOD.reprocess() — public API for external trigger after filters
  Init on DOMContentLoaded + retries at 1.5s + 4s for async renders
